# WhatsApp 2.0

## Project Structure
- **src/**: Contains the source code for the application.
- **public/**: Contains public assets such as images and stylesheets.
- **tests/**: Contains unit and integration tests.
- **docs/**: Contains documentation files.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/CraftyPlayerYT/WhatsApp-2.0.git
   cd WhatsApp-2.0
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the application:
   ```bash
   npm start
   ```

## Architecture
The application follows an MVC architecture, separating the presentation layer from the business logic.
- **Model**: Represents the data and business logic.
- **View**: Represents the UI components.
- **Controller**: Handles user input and interacts with the model to produce outputs.
